[HFR] Filtre A&V
================

Permet de filtrer les topics des catégories Achats & Ventes.

## Fonctionnalités
- Filtrage des achats
- Filtrage des ventes
- Filtrage des échanges

## Compatibilité

- Navigateurs : 
  - Firefox 82.0.1
  - Chrome - non testé
- Extensions :
  - Greasemonkey - non testé
  - Tampermonkey - non testé
  - Violentmonkey v2.12.7
